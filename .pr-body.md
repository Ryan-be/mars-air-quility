## Deploy checklist

For a fresh MLSS server install: `bash scripts/setup_pi.sh` (handles apt deps incl. ffmpeg, I2C, UART, dialout group, poetry, certs, DB init) then `sudo systemctl enable --now mlss-monitor`.

For an existing MLSS server upgrading to this branch: `git pull && sudo systemctl restart mlss-monitor`. If hitting `EACCES` on `/dev/serial0`, additionally `sudo usermod -aG dialout masadmin && sudo raspi-config nonint do_serial 2 && sudo systemctl disable --now serial-getty@ttyAMA0.service && sudo reboot`.

For grow units: flash the SD-card image from `scripts/build_pi_image.sh`, drop `mlss-grow.yaml` on the boot partition with MLSS hostname + enrollment key, first-boot script handles the rest.
